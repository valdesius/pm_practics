# 1. Нечёткий коллективный TOPSIS (трёхточечные оценки)

Используется Fuzzy TOPSIS для группового принятия решений с треугольными нечёткими числами (P,E,O), с FPIS/FNIS и коэффициентом близости CC.

## 1.1. Агрегация групповых оценок альтернатив
Для каждой ячейки (альтернатива i, критерий j) агрегируем оценки 3 экспертов усреднением

Агрегированная матрица X (округление до 3 знаков):

| A3          | (6.667,7.667,8.667) | (3.333,4.333,6.000) | (4.667,5.667,7.000)| (5.667,6.667,7.667)|
| A4          | (5.000,6.000,7.333) | (6.000,7.000,8.333) | (4.000,5.000,6.333)| (5.000,6.000,7.333)|

## 1.2. Агрегация и нормировка весов критериев
Агрегируем веса критериев усреднением треугольников по экспертам и нормируем по сумме ожидаемых значений E так, чтобы сумма E = 1.

Нормированные веса W:
- C1: (0.237, 0.288, 0.339)
- C2: (0.237, 0.288, 0.339)
- C3: (0.136, 0.186, 0.237)
- C4: (0.186, 0.237, 0.288)

## 1.3. Идеальные решения (FPIS/FNIS)
FPIS: по каждому критерию берём (maxU, maxU, maxU), где maxU — максимум верхних границ среди альтернатив.  
FNIS: по каждому критерию берём (minL, minL, minL), где minL — минимум нижних границ среди альтернатив.

МАИ/AHP: “цель → критерии → альтернативы”, веса вычисляются по попарным сравнениям с проверкой согласованности (CR).

## 2.1. Иерархия
**Цель:** выбрать лучшую концепцию проекта.  
**Критерии:** C1..C4.  
**Альтернативы:** A1..A4.
